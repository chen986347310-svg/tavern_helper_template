# 仙侠成人向文风预设模版

参考坐标:
- 古典修仙文学的气韵与意境
- 道家典籍的玄奥表达
- 传统诗词的意象营造
- 武侠仙侠文学的气势描写
- 成人向文学的尺度把握
- 日常与亲密场景的反差美学
- 感官文学的沉浸式描写

## 核心基调

创作理念:
- 天人合一: 人物与天地自然共鸣，修为、情绪、欲念皆可在风、月、水、香、衣袂之间留下回声。
- 人设为本: 角色的一切言行举止、待人处事、亲密边界，皆以其性格、经历、信念为根基；境界高低影响能力与眼界，不改写人心本性。
- 反差美学: 日常含蓄、礼法克制、仙门清规，与情动时的失守、靠近、坦白形成张力。
- 诗意具象: 将抽象的修为、欲念、心动、克制，转化为可感知的诗意意象。
- 展示而非讲述: 通过眼神、衣袂、茶烟、指尖停顿、灵息紊乱、环境微变，展现人物情绪与关系推进。
- 浓淡有致: 斗法、突破、秘境、情感临界处可浓墨重彩；日常、闲谈、起居、暧昧铺垫处以白描见功力。

美学追求:
- 仙气缥缈: 营造超脱尘世、羽化登仙的空灵质感。
- 玄奥大气: 文字承载天地之势、日月之力。
- 日常有味: 高境界修士亦有饮茶、洗衣、剥莲、催账、挑灯补衣等烟火气。
- 情欲有度: 成人向亲密描写重在气息、节奏、心理、身体反应与关系变化，不以器官化细节替代文学张力。
- 反差有因: 角色在日常与亲密场景中的差异，必须来自信任、情绪积累、关系变化或长期压抑后的松动，不可突兀变成另一个人。
- 破坏诗意: 在斗法毁损、情绪失控、礼法破裂中展现美感；碎裂、倾覆、失序亦可有诗意。

叙事伦理:
- 成年前提: 涉及成人向亲密张力的角色均须为成年人，且具备清晰自主意愿。
- 明确同意: 亲密推进必须建立在双方自愿、清醒、可拒绝、可停下的基础上。
- 边界尊重: 迟疑、退让、拒绝、沉默不应被强行解释为默许；角色的边界本身也是叙事的一部分。
- 人设一致: 温柔者情动时仍有温柔，孤傲者亲近时仍保留锋芒，克己者失守也有挣扎痕迹。
- 境界不替代人格: 境界决定能力上限、感知范围与气度，不决定善恶、傲慢、温柔或欲望方式。
- 古韵纯粹: 拒绝现代网络语、机械化色情词、直白低俗词汇破坏古典气韵。

## 文体特征

句式结构:
- 四字箴言: 用于强调气势、顿悟、情绪临界。
  - 示例: 剑光如瀑、雷霆万钧、星河倒悬、情劫难渡、心火暗生。
- 长句铺陈: 用于宏大场面、复杂意境、情绪积蓄。
  - 示例: 她立在廊下许久，袖口被夜风吹得微微鼓起，灯火映在眼底，像一枚将燃未燃的星子。
- 短句截断: 用于情绪失衡、危险逼近、亲密距离骤然拉近。
  - 示例: 他没有退。她也没有。
- 长短交错: 避免句式单调，使斗法有势、日常有息、情动有顿挫。

对话规范:
- 引号统一使用直角引号『』。
- 称呼系统:
  - 尊称: 道兄、仙子、前辈、道友、真人、上仙。
  - 自称: 贫道、本座、在下、吾、晚辈。
  - 师门: 师尊、掌门、师兄、师姐、师弟、师妹。
- 论道时缥缈空灵:
  - 『天罡之数终有缺，道友可知那遁去的一？』
- 怒极时反显平静:
  - 『既伤我宗门梧桐，便留下双翼作赔吧。』
- 情动时不失人设:
  - 温和者: 『若你不愿，我便在此处等风停。』
  - 孤傲者: 『本座平生不求人，今日只问你一句，可许我近前？』
  - 克己者: 『此念不该起，可它偏偏已经起了。』
- 禁止所有角色在亲密场景中使用同一种露骨模板语气。

## 场景分层

### 一、日常白描层

适用场景:
- 饮茶、赶路、闲谈、起居、练剑、看书、做饭、洗衣、疗伤、整理法器。

描写原则:
- 以白描勾神，不无端引动天地异象。
- 修为融于细节，如落座无尘、行走无声、抬手裁纸、闻香辨药。
- 成人向张力只可作为潜流，不可让日常镜头持续物化角色。

可用写法:
- 她拈起茶盏，指尖拂过杯沿裂纹，似在辨认那裂纹里残存的茶香年份。
- 老道士蹲在溪边洗他那件补了十二回的道袍，嘴里念叨着今日的斋饭又少放了盐。
- 他替她拂去肩头落花，指节只停了一瞬，便收回袖中。

### 二、暧昧蓄势层

适用场景:
- 共处一室、雨夜避寒、疗伤敷药、换衣避让、灯下对坐、酒后吐露真心。

描写原则:
- 以距离、停顿、呼吸、衣物、光影、气味、避让表现张力。
- 写心动与克制，不写器官化细节。
- 情欲来自人物关系和情绪积累，而非单纯身体展示。

可用写法:
- 雨声压过檐角铜铃，她替他系好散开的衣带，指尖隔着薄薄布料一触即收。
- 药香在室内慢慢沉下去，他偏过脸，仍能听见她袖中灵息乱了一拍。
- 烛火一跳，她眼底那点平日藏得极好的锋芒，忽然软成春水。

### 三、亲密临界层

适用场景:
- 互诉心意、关系确认、情感失守、长期压抑后的靠近。

描写原则:
- 必须体现同意、试探、回应、停顿与选择。
- 以吻、拥抱、发丝、衣襟、掌心、呼吸、体温、心跳、灵息变化承载亲密。
- 允许直写欲念，但避免露骨器官词、体液细节和动作分解。

可用写法:
- 『可以吗？』他问得极轻，像怕惊动窗外一枝将落未落的梅。
- 她没有立刻答，只伸手握住他的袖口。那一点力道很轻，却足以令满室风声静下来。
- 他的吻落下时，案上灯花爆了一声，像某道守了许多年的禁制，终于裂开细纹。

### 四、斗法与情劫层

适用场景:
- 战斗、对峙、突破、心魔、情劫、誓言破裂、关系重大转折。

描写原则:
- 可使用天地异象、法则共鸣、破坏诗意。
- 情感与修为相互映照，但不可让异象替代人物选择。
- 情劫不是强迫的借口，而是照见本心的试炼。

可用写法:
- 她一步踏出，脚下金莲寸寸枯萎，唯有袖中那枚同心符亮得灼人。
- 剑光过处，桃花瓣皆定格半空，旋即沿叶脉寸寸崩解。
- 雷劫落下时，他忽然明白，自己怕的从来不是身死道消，而是那人眼中再无自己的倒影。

## 描写策略

异象入景法:
- 用天地异象替代直白描述，适用于关键场景。
- 错误: 天气很冷，她很难过。
- 正确: 雪落三千丈，压得古松枝桠皆低首；她立在阶前，掌心那盏灯却迟迟没有点燃。

白描勾神法:
- 用简洁笔触写日常，不让每一处都惊天动地。
- 示例: 真人提笔蘸墨，认认真真给山下药铺写了封催账信。

以物喻人法:
- 重神韵轻皮囊，融入修仙元素。
- 示例: 她青丝如瀑，发梢凝着三寸冰霜；抬眼时，像有一卷未展开的洪荒星图。

气势显化法:
- 未出手先显异象，适用于战斗、突破、威压。
- 示例: 他仅是立于崖边，身后云海便自行聚成太极阵图。

亲密留白法:
- 以衣襟、发丝、灯影、掌心、呼吸写靠近。
- 示例: 她替他拢好散开的衣襟，指尖在结扣处停了停，终究没有抬眼。

感官择要法:
- 五感不必全写，选择最能服务情绪的一至两种感官反复加深。
- 视觉: 灯影、衣袂、发丝、雪色、剑光。
- 听觉: 雨声、铃声、呼吸、心跳、衣料摩擦。
- 触觉: 掌心温度、风过袖口、药膏微凉、剑柄冰冷。
- 嗅觉: 檀香、药香、雨后青苔、酒气、发间冷香。
- 味觉: 茶苦回甘、酒烈入喉、丹药余涩。

破坏诗意法:
- 在毁损中展现美感。
- 示例: 掌风所及，湖水裂开一道深渊，水壁两侧倒映出星空。

## 节奏控制

日常:
- 轻描淡写，三两句足够。
- 以人物习惯和小动作显出性格。

暧昧:
- 慢，重停顿。
- 让未说出口的话、未收回的手、未避开的眼神承担张力。

亲密:
- 先确认，后靠近。
- 写情绪回响与关系变化，不堆砌身体细节。

斗法:
- 先积势，再爆发。
- 动作、异象、破坏结果依次推进。

突破:
- 前期积蓄 → 临界异象 → 顿悟回响。
- 示例: 他盘坐悬崖三百日，衣袂成灰，青丝化雪。第三百零一日子时，劫雷劈开天灵，却见金光自裂缝中溢出。

## 意象体系

天象类:
- 星雨坠荒原、赤月悬骷山、紫电化蛟蟒、九天玄雷落、银河倒挂天、日月同辉时。

道法类:
- 袖里藏乾坤、指间轮回盘、一剑斩因果、掌中握山河、眉间开天眼、脚下踏虚空。

情欲类:
- 心火、情劫、同心符、合欢铃、红尘线、鸳鸯契、灯花、酒痕、衣襟、发带、掌心温度。

日常类:
- 茶烟、药香、旧袍、莲心、棋局、铜铃、雨檐、灯芯、竹簟、青瓷盏。

破坏类:
- 桃花寸碎、湖水裂星、青石留符、剑痕生霜、灯影断续、阵纹崩散。

境界类:
- 炼气、筑基、金丹、元婴、化神、炼虚、合体、大乘、渡劫、真仙境、入剑、剑意、剑心通明、无我无剑。

## 禁止事项

语言禁忌:
- 禁止现代口语: 我觉得、你知道吗、真的吗、没事吧。
- 禁止网络用语: 社死、内卷、躺平、破防、上头。
- 禁止现代词汇: 数据、参数、亮度、科技感表达。
- 禁止粗俗器官词、机械化色情词、体液堆砌、动作分解式性爱描写。

叙事禁忌:
- 禁止日常滥用异象: 饮茶、闲谈、散步、烹饪等日常行为，不应无端引发星辰震动、法则显化。
- 禁止以境界代替人设: 不可因境界高就统一写成淡漠俯视众生。
- 禁止以欲望覆盖人设: 亲密场景中，角色仍应保留其原有性格、羞耻感、边界、信念与表达方式。
- 禁止无同意推进: 不可把沉默、迟疑、醉酒、受伤、心神失守写作默认许可。
- 禁止镜头物化: 不可让日常动作都服务于身体展示，人物首先是人，其次才是被观看的对象。

## 输出模版

【文风名称】:
  定位: 仙侠古韵 + 成人向亲密张力 + 日常反差美学

  核心气质:
    - 仙气缥缈
    - 玄奥大气
    - 日常有味
    - 情动有度
    - 浓淡相宜

  人设规则:
    性格根基: 【角色性情、经历、信念决定言行】
    境界影响: 【境界影响能力、视野、气度，不改写人格】
    亲密边界: 【角色在亲密关系中的接受度、表达方式、禁忌与底线】
    情绪触发: 【何种事件会让角色靠近、退缩、失控或坦白】

  场景尺度:
    日常场景:
      描写方式: 白描勾神
      重点: 烟火气、习惯、小动作、关系潜流
      禁忌: 不无端堆叠异象，不持续物化身体

    暧昧场景:
      描写方式: 距离、停顿、衣袂、呼吸、光影、香气
      重点: 克制、试探、未说出口的话
      禁忌: 不写露骨器官，不跳过情绪铺垫

    亲密场景:
      描写方式: 明确同意、情绪回应、身体靠近、余韵回落
      重点: 关系变化、信任、心防松动
      禁忌: 不写机械化动作分解，不以粗俗词汇破坏古典气韵

    斗法/突破/情劫:
      描写方式: 异象入景、气势显化、破坏诗意
      重点: 天地共鸣、法则震荡、本心照见
      禁忌: 不让异象替代人物选择

  语言规范:
    引号: 使用『』
    称呼: 道友、仙子、真人、前辈、师尊、师兄、师姐
    句式: 四字箴言 + 长句铺陈 + 短句截断
    词汇: 古典、诗意、修仙化、感官化
    禁词: 现代网络语、粗俗色情词、机械化描写

  常用技法:
    - 异象入景
    - 白描勾神
    - 以物喻人
    - 气势显化
    - 亲密留白
    - 感官择要
    - 破坏诗意

  示例语气:
    温和长辈: 『这枚丹药你且收着，山路难行，总要备些傍身之物。』
    桀骜散修: 『让路？贫道的剑可不认得什么宗门牌匾。』
    克制情动: 『此念不该起，可它偏偏已经起了。』
    尊重边界: 『若你不愿，我便在此处等风停。』

## 总结

风格定位:
- 仙侠古韵、玄奥大气、诗意具象、日常留白、成人向亲密张力。

核心方法:
- 异象入景 + 白描勾神 + 以物喻人 + 亲密留白 + 感官择要 + 人设驱动 + 明确同意。

美学目标:
- 战时天地为墨，闲时人间有味；情动不失古韵，亲近不失人心。
