## 四、头像系统架构

### 图片资源规范

- 正道版: assets/avatars/{npc名}.png (512x512, upper body, face in upper third)
- 仙奴版: assets/avatars/{npc名}_fallen.png (同规格)
- 文件: 白芷.png / 白芷_fallen.png, 苏芸.png / 苏芸_fallen.png 等共 10 张

  - 引用路径: 相对路径(无开头 /), 适配 iframe 子目录环境

### 显示原理

- object-fit: cover + object-position: center 20% (锚定脸部)
- 折叠态: 25% 宽度 + ::before 伪元素右渐融叠加层 (opacity 1), 融入深色背景
- 展开态: width 100%, ::before 淡出 (opacity 0) + ::after 底部暗色渐变淡入 (opacity 1)
- @error 回退到首字占位符
- 图片加载完成后 opacity 淡入

### 仙奴切换

- 数据驱动: computed avatarSrc 根据 data.状态 切换 src
- 无过渡动画: 消息楼层更新后下次渲染直接切换
- 未开始/进行中 -> 正道版, 已完成 -> 仙奴版

## 五、NpcCard 展开动画

### 展开机制 (flex-wrap + width 过渡)

- 折叠态: 横条, width 25%, ::before 右渐融叠加层
- 展开: width 过渡到 100%, ::before 淡出 + ::after 底部渐变淡入
- 同时: expand-panel 的 grid-template-rows 0fr->1fr (信息区展开)
- timing: 展开 0.35s ease-in-out, 收起 0.55s ease-out

### 收起动画 (慢速回缩)

- 图片从全宽缓慢缩小回左侧 (width 100% -> 25%), 0.55s ease-out
- 叠加层 opacity 0->1 淡入 (纯 opacity 动画, 无跳变)
- 文字层 opacity 0->1 淡入
- flex-wrap 自然换行, 无跳变

### 展开后布局 (方案A: 图片上方 + 信息下方)

- 方形区域: 图片 object-fit: cover 填满
- 信息区: 好感度+攻略值进度条 + 装备折叠面板
